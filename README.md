# Animated To-Do List

A beautiful, animated To-Do List web app built with HTML, CSS, and JavaScript.  
Tasks are saved in your browser (localStorage), and you can mark them complete or delete them with a click.

![Screenshot](logo.png)

## 🚀 Features

- Add, complete, and delete tasks with smooth animations
- Tasks are saved even after page reload (localStorage)
- Responsive and modern UI with animated transitions
- Pure front-end (no backend required)

## 🛠️ Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Start adding tasks!

## 🖥️ How it works

- **Add Task:** Type in the input and click "Add" or press Enter.
- **Complete Task:** Click the ✓ checkmark or on the task to mark as complete/incomplete.
- **Delete Task:** Double-click a task or click the ❌ cancel icon to remove.
- **Persistence:** Tasks are saved in your browser using `localStorage`.

## 📢 Customization

- **Colors & Animations:** Easily edit `tdl.css` to change the look and feel.
- **Logo:** Replace `logo.png` with your own image if you like.
